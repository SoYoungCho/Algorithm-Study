## 나의 풀이

```python
def solution(seoul):
    return "김서방은 "+ str(seoul.index('Kim')) + "에 있다"

solution(["Jane","Kim"])
```

리스트에서 해당 요소의 인덱스 구하기 : _리스트_.index(_찾고자 하는 요소_)  
index() 함수만 알면 쉽게 풀 수 있는 문제이다!

## 모범 답안 (나의 풀이와 비슷)

```python
def findKim(seoul):
    return "김서방은 {}에 있다".format(seoul.index('Kim'))

print(findKim(["Queen", "Tod", "Kim"]))
```

{} , 그리고 .format()을 사용해 비슷하게 풀었다.
